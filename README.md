# SIMULOGIC: Gamificação de Circuitos Digitais

> **Status do Projeto:** 🛠️ Em Desenvolvimento (Acompanhamento Acadêmico)

## 💡 Sobre o Projeto
O **Simulogic** é uma ferramenta educacional interativa projetada para simplificar o aprendizado de lógica digital. O projeto foca na criação de um ambiente onde estudantes podem visualizar, montar e testar circuitos lógicos de forma intuitiva, unindo teoria e prática através de um sistema gamificado.

## 🎯 Objetivos
- **Educacional:** Facilitar a compreensão de portas lógicas e fluxos de sinais.
- **Prático:** Permitir a simulação de circuitos sem a necessidade de hardware físico.
- **Engajamento:** Utilizar um módulo de níveis (jogo) para incentivar a progressão do aprendizado.
